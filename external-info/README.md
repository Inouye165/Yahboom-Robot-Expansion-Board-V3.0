# External Information Directory (`external-info/`)

This directory is the storage hub for raw reference materials, downloaded documentation, web screenshots, and research notes.

## Directory Structure

* **[official-docs/](file:///c:/Users/inouy/electronics-project-hub/driver-boards/Yahboom_Robot_Expansion_Board_V3.0/external-info/official-docs/)**
  * Purpose: Storing downloaded Yahboom PDFs, schematics, manuals, or specification notes.
  * *Policy:* If licensing or copyright is unclear, prefer capturing links and summarization notes in this repo rather than committing large proprietary binary documents.
* **[screenshots/](file:///c:/Users/inouy/electronics-project-hub/driver-boards/Yahboom_Robot_Expansion_Board_V3.0/external-info/screenshots/)**

  * See the [Screenshots README](file:///c:/Users/inouy/electronics-project-hub/driver-boards/Yahboom_Robot_Expansion_Board_V3.0/external-info/screenshots/README.md) for naming guidelines.
* **[amazon-listing/](file:///c:/Users/inouy/electronics-project-hub/driver-boards/Yahboom_Robot_Expansion_Board_V3.0/external-info/amazon-listing/)**
